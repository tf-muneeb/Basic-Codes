# Basic C/C++ Codes

This repository contains a collection of beginner-friendly C/C++ programs created for practice and learning programming fundamentals.

## Programs Included
- Pattern printing
- Number system conversions
- Word search
- Histogram generation
- Obstacle problem
- String manipulation
- Hotel Management System (console-based project)

## Technologies Used
- C/C++
- Object-Oriented Programming (basic concepts)
- Problem-solving techniques

## How to Run
1. Open file in any C/C++ IDE (Dev C++, VS Code, CodeBlocks)
2. Compile program
3. Run executable

Example:
g++ filename.cpp
gcc filename.c
./a.out

## Purpose
This repository is useful for:
- Beginners learning C/C++
- Practicing logic building
- Understanding programming basics
- Academic assignments

- Loops
